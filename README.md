# E-Volve

A modern React web application built with Vite, styled for visual excellence, and animated using Framer Motion.

## 🛠️ Tech Stack & Dependencies

Here is the tech stack used in this project:

### Core Framework & Build Tool
*   **[React 18](https://react.dev/)** - A JavaScript library for building user interfaces.
*   **[Vite 5](https://vitejs.dev/)** - A fast build tool and development server.

### Routing & Navigation
*   **[React Router DOM 6](https://reactrouter.com/)** - Declarative routing for React applications, enabling seamless page navigation.

### Styling & Animations
*   **Vanilla CSS** - Premium and flexible styles tailored for high-quality aesthetics.
*   **[Framer Motion](https://www.framer.com/motion/)** - A production-ready motion library for React, powering smooth transitions and micro-interactions.

### Icons & Utilities
*   **[Lucide React](https://lucide.dev/)** - A clean and consistent icon library.

### Development & Linting
*   **ESLint** - Code quality tool to maintain clean, consistent JavaScript/React code.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### Installation
Clone the repository and install the dependencies:

```bash
# Install dependencies
npm install
```

### Running Locally
To start the development server:

```bash
npm run dev
```

### Building for Production
To build the application for production:

```bash
npm run build
```
