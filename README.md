# E-Volve

A modern React web application built with Vite, styled for visual excellence, and animated using Framer Motion.

## 🛠️ Tech Stack Architecture

The application is structured into the following layers:

### 💻 Frontend (FE)
*   **Framework**: [React 18](https://react.dev/) - Component-based user interface library.
*   **Build Tool**: [Vite 5](https://vitejs.dev/) - Next-generation front-end tooling for extremely fast development.
*   **Routing**: [React Router DOM 6](https://reactrouter.com/) - Declarative navigation and page routing.
*   **Animations**: [Framer Motion 11](https://www.framer.com/motion/) - Dynamic interactive animations and smooth page transitions.
*   **Icons**: [Lucide React](https://lucide.dev/) - Clean and consistent modern vector icons.
*   **Styling**: Premium Vanilla CSS custom styles.
*   **Linting**: ESLint for maintaining code quality.

### ⚙️ Backend (BE)
*   **Current State**: Client-side state simulations and mock database.
*   **Planned Tech**: Node.js & Express.js for building scalable REST API endpoints.

### 🤖 AI Integration
*   **Model**: **Google Gemini (Gemini 1.5 Flash / Pro)**
*   **Capabilities**:
    *   **Gemini Vision**: Identifies device models, serial numbers, and inspects physical components from images.
    *   **Teardown Guide Generation**: Generates step-by-step instructions for safely dismantling electronic parts.
    *   **Smart Pricing Suggestions**: Estimates market value of components to facilitate circular economy trade.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### Installation
Clone the repository and install the dependencies:

```bash
# Install dependencies
npm install
```

### Running Locally
To start the development server:

```bash
npm run dev
```

### Building for Production
To build the application for production:

```bash
npm run build
```
